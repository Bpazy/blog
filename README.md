# Usage
Node.js and hexo is required. Install node.js then `npm install -g hexo`.  
## Deploy 
Use: `hexo g -d`   
If deploy wrong: `rm -rf .deploy_git public && hexo g -d`    
## New article
Use: `hexo new [layout] article-title`
